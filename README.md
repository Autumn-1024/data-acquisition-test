# 采集卡测试工程

STM32F103C8T6 传感器数据采集测试 — OLED 显示 + 串口控制

## 硬件配置
- **MCU**: STM32F103C8T6
- **OLED**: SSD1306 128x64, 软件I2C (PB8=SCL, PB9=SDA)
- **按键**: KEY1=PB10, KEY2=PB11, KEY3=PA11, KEY4=PA12 (低电平有效)
- **调试串口**: USART1 PA9/PA10, 115200bps
- **LED**: PC13 (低电平点亮)

## 功能说明
- 开机显示欢迎界面，任意按键进入传感器列表
- 10 个传感器 (Sensor 1 ~ Sensor 10)，上下键选择，KEY4 进入详情
- 传感器详情页：传感器名称 + 大字数据值(cm) + 实时波形图，每秒刷新
- 数据范围 1.0 ~ 20.0，一位小数
- 当前为模拟数据模式，后续接入真实传感器替换

## 按键映射
- **KEY1 (PB10)** → 返回
- **KEY2 (PB11)** → 下移
- **KEY3 (PA11)** → 上移
- **KEY4 (PA12)** → 确认 / 进入

## 串口命令 (115200bps)

### 传感器跳转
| 命令 | 功能 |
|------|------|
| `1` ~ `9` | 跳转到 Sensor 1 ~ 9 |
| `10` | 跳转到 Sensor 10 |

### LED 控制
| 命令 | 功能 |
|------|------|
| `a` | LED 亮 |
| `b` | LED 灭 |
| `c` | LED 翻转 |
| `d` | 查询 LED 状态 + 当前传感器序号 |

### 数据输出
每秒自动发送当前传感器数据，格式：`传感器序号@数据值`，例如 `1@9.6`
